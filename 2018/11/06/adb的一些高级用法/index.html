adb的一些高级用法

发表于 2018-11-06
查看包名list

adb shell pm list package

查看内存情况

adb shell cat /proc/meminfo

查看CPU及内存使用情况

adb shell top

查看应用内存占用

adb shell dumpsys meminfo

卸载应用

adb shell pm uninstall –user 0 com.yunos.tv.advertise.pushcenter

存log 带time

adb shell logcat -v time 2>&1 | tee logtime.log
